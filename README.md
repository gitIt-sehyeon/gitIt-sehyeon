### Featured Project – UsFit  
#### **스포츠 기반 매칭 & 커뮤니티 플랫폼**

UsFit은 운동을 통해 사람들을 연결하는 스포츠 플랫폼으로,  
**주변 체육시설 탐색 → 동호회 → 용병 매칭 → 리뷰 → 활동 기록**까지 한 번에 제공하는 서비스입니다.

- Backend Repo: https://github.com/Us-Fit/Us-Fit-BE  
- Frontend Repo: https://github.com/Us-Fit/Us-Fit-FE  

#### Key Features
- 위치 기반 체육시설 탐색  
- 운동 동호회 목록 / 상세 / 가입  
- 용병 모집 및 참여  
- 운동 활동 생성 및 후기 작성  
- S3 Presigned URL 이미지 업로드  
- JWT 기반 인증  
- CloudWatch 로그 모니터링  
- 추천 기반 활동 탐색
